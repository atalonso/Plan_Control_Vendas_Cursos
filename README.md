# Plan_Control_Vendas_Cursos
Este projeto consiste no desenvolvimento de um dashboard de vendas de cursos utilizando o Microsoft Excel, com o objetivo de apresentar os dados de forma visual, clara e de fácil interpretação. O dashboard foi criado para auxiliar na análise do desempenho dos cursos, permitindo identificar o volume de vendas e o valor total arrecadado a partir das informações disponíveis. Os dados utilizados são fictícios e representam registros de vendas de diferentes cursos, como Excel Avançado, Excel Essencial, Gestão Financeira, Marketing Digital e Power BI, incluindo o status da venda e os valores correspondentes.

A proposta do dashboard é facilitar a tomada de decisão por meio de gráficos e filtros interativos, possibilitando ao usuário visualizar rapidamente os resultados e comparar informações conforme o curso selecionado. A estrutura foi organizada de maneira simples e intuitiva, utilizando tabelas e gráficos dinâmicos do Excel, além de segmentações de dados que permitem explorar diferentes cenários. Este projeto foi desenvolvido como parte de um desafio prático da DIO, aplicando conceitos básicos de análise de dados e visualização, demonstrando como o Excel pode ser uma ferramenta eficiente para transformar dados em informações relevantes.
